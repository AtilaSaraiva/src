Detailed Description

template<typename Scalar, typename LSPolicy, typename LSPolicyData>
class RVLUmin::LinFitLS< Scalar, LSPolicy, LSPolicyData >

Given an Operator F and a Vector d in the range of op, implements the function $ as an RVL::Functional.

The linear least squares solver is specified by policy.
